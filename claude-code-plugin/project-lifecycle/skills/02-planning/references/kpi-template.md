## 2. SMART 기준 검증

각 KPI가 SMART 기준을 충족하는지 확인:

| 기준 | 설명 | 체크 |
|------|------|------|
| **S**pecific | 구체적으로 정의되었는가? | [ ] |
| **M**easurable | 측정 가능한가? | [ ] |
| **A**chievable | 달성 가능한 목표인가? | [ ] |
| **R**elevant | 프로젝트 목표와 관련 있는가? | [ ] |
| **T**ime-bound | 기한이 있는가? | [ ] |

## 5. Phase 간 추적성

> 이 KPI는 다음 Phase에서 참조된다:
> - **Phase 7 (QA)**: 인수 테스트(Acceptance Test)의 합격 기준으로 사용
> - **Phase 8 (운영/회고)**: 회고 시 달성률 측정 기준으로 사용
> - **Dashboard**: 프로젝트 건강도 지표로 표시

*KPI는 프로젝트 진행에 따라 업데이트될 수 있으며, 변경 시 이해관계자의 동의를 거친다.*
